# Student-performance-analysis

## Tools Used
- Microsoft Excel/Metabase
- Pivot Tables
- Charts & Graphs
- Data Cleaning Techniques

## Dataset Description
The dataset contains:
- Student ID
- Name
- Course Code
- Course Name
- Test score
- Exam Score
- Attendace
- Total Score
- Grade
- Remark
- Status

## Data Cleaning Process
-Removed Duplicate
-Handled missing values
-Standardized data format
-Oranized data for analysis

## Dashboard Features
- Student by Average score
- Toytal score vs Attendance
- Average score by Course
- Pass vs Fail Distribution
- Grade Distribution

## Key Insights
- Students with attendance above 90% performed better overall.
- Networking had the highest average score.
